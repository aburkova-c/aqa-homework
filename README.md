Домашние задания курса AQA
Merge conflict — это ситуация, когда Git не может автоматически объединить изменения из разных веток, 
потому что они затрагивают один и тот же участок кода или файла. 
Для разрешения конфликта нужно открыть конфликтный файл, выбрать или объединить нужные изменения, 
убрать конфликтные маркеры, выполнить git add и завершить merge или rebase.
